Run PowerShell Setup

One command. Copy, paste into PowerShell (as Administrator), and you're done.

irm https://onboarding.pspace/assets/install-windows.ps1 | iex
1

Press Win + X, then click Windows PowerShell (Admin)

2

Copy the command above and paste it into PowerShell

Right-click to paste in PowerShell
3

Press Enter. You may see a security prompt — type Y and press Enter

The script will show "PaceySpace certificate installed successfully"

Prefer manual install? Download the certificate and:

  1. Double-click the downloaded file
  2. Click "Install Certificate"
  3. Select "Local Machine" → "Next"
  4. Select "Place all certificates in the following store"
  5. Click "Browse" → select "Trusted Root Certification Authorities"
  6. Click "Next" → "Finish"
Recommended

Run Terminal Setup

One command in your terminal. Works on Ubuntu, Debian, Fedora, and most distributions.

curl -fsSL https://onboarding.pspace/assets/install-linux.sh | sudo bash
1

Open your terminal (Ctrl+Alt+T on Ubuntu)

2

Copy the command above and paste it into the terminal

3

Enter your password when prompted. The script will confirm installation.

Prefer manual install? Download the certificate and run:

sudo cp ~/Downloads/paceyspace-ca.crt /usr/local/share/ca-certificates/ && sudo update-ca-certificates
