🌐 DNS not resolving? Click here for help

Your Mac is connected to Tailscale, but it needs a Split DNS rule in Tailscale's admin console to know that *.pspace domains should be sent to our AdGuard DNS server (not MagicDNS).

Ask your admin to add this rule:

  1. Open Tailscale Admin → DNS
  2. Scroll to Split DNS
  3. Click Add nameserver → use domain pspace → nameserver 100.121.252.126
  4. Wait ~30 seconds, then click Run Tests again

Once added, you'll resolve *.pspace automatically. This is a one-time setup that all colleagues need.
